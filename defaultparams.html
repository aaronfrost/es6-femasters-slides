Assign w/ Method Call

  •   function getRand(){
    return Math.ceil(Math.random() * 10000000) + new Date().getTime();
  }

  function myFunction(id=getRand()){
    console.log("My ID: "+id);
  }

  myFunction(); //Logs random number
  myFunction(1); //Logs 1

Assign w/ Method Call

  •   function die(){ throw "x"}

  function test(a = die()){
    console.log("Didn't die");
  }

  test(); // throws an error
  • Errors from a default assignment function will propagate
